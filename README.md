a bukkit plugin.
## Usage
### Commands
#### /atm
args:
+ open - ATM open commands.
+ help - Help view commands.
+ reload - @admin (Config/lang reload commands.)
### Permissions
#### atm.open
##### ATM Open permissions
###### Commands:
* /atm open
#### atm.admin
##### ATM Admin permissions
###### Commands:
* /atm reload
* /atm help
